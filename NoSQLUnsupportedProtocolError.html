NoSQLUnsupportedProtocolError

NoSQLUnsupportedProtocolError

This error indicates that the server is running at a lower protocol version than the client (i.e. the client is using a newer protocol version than the server supports). The client should attempt to decrement its internal protocol version and try again. Errors of this class have error code ErrorCode.UNSUPPORTED_PROTOCOL.

Indicates whether this error is retryable.

APIs that result in retryable errors are automatically retried by the driver's default RetryHandler. Default retry handler can be customized by properties in RetryConfig. Alternatively, a custom RetryHandler can be set as RetryConfig#handler.

If necessary, APIs that result in retryable errors may also be explicitly retried by the application.
